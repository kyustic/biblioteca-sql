# Sistema de Biblioteca – Scripts SQL

Este projeto faz parte da atividade prática de Banco de Dados. Aqui estão os scripts usados para criar o banco, montar as tabelas e fazer testes de consultas e manipulação de dados no MySQL Workbench.

## Arquivos do projeto
- **create_tables.sql** – cria o banco e todas as tabelas.
- **insert_data.sql** – insere os primeiros dados nas tabelas.
- **selects.sql** – consultas usando JOIN, WHERE, ORDER BY e LIMIT.
- **updates.sql** – comandos de atualização de dados.
- **deletes.sql** – comandos de remoção seguindo as regras de integridade.


## Sobre o projeto
O banco foi feito com base no minimundo de uma biblioteca, contendo as tabelas:
- Usuario  
- Categoria  
- Livro  
- Emprestimo  
- Avaliacao
